Logistics Management System

The Logistics Management System is a backend-based web application developed using Java and Spring Boot to make transportation and delivery management easier and more organized. This project was built to simulate how logistics companies manage orders, drivers, trucks, and delivery operations in real time.

The application includes both Admin and User modules with separate functionalities. Users can create delivery orders, assign drivers, and check delivery details, while admins can manage trucks, drivers, carriers, and monitor overall logistics activities. The system is designed to reduce manual work and improve the efficiency of delivery management.

For backend development, technologies like Spring Boot, Spring MVC, Spring Data JPA, Hibernate, and MySQL were used. REST APIs were implemented for handling communication between different modules, and CRUD operations were performed for managing orders, vehicles, and user data. The project follows a layered architecture, making the code clean, maintainable, and easy to understand.

Working on this project helped in gaining practical experience with real-world backend development concepts such as API creation, database connectivity, entity relationships, exception handling, dependency injection, and role-based access control. Tools like Postman, Maven, Git, and GitHub were also used during development for API testing, dependency management, and version control.

Overall, this project reflects my understanding of Java Full Stack development and my ability to build scalable backend applications using Spring Boot and modern development practices.
